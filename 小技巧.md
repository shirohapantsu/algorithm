# 算法竞赛小技巧

## 一、IO优化

### 1.关闭同步流

```cpp
ios::sync_with_stdio(false);    // 解除C++流与C标准流的绑定
cin.tie(nullptr);               // 解除cin和cout的绑定
cout.tie(nullptr);
```
注意在使用了这串代码后，不能在程序中混用cin/cout和scanf/printf

### 2.避免使用endl

`std::endl` 不仅会换行，还会强制刷新输出缓冲区（flush）。在大量输出时，这会极其缓慢。所以最好使用`\n`替换`endl`

## 取整a/b

```cpp
res = a / b;    // 默认就是向下取整
res = (a + b - 1) / b;  // 向上取整
res = (a + b / 2) / b;  // 四舍五入
```

## 二、前缀和

缀和可以简单理解为「数列的前 $n$ 项的和」，是一种重要的预处理方式．对于长度为 $n$ 的序列 $\{ a_i\}$ ，如果要多次查询区间 $[l,r]$ 中序列数字的和，就可以考虑使用前缀和．

要想查询区间 $[l,r]$ 内的序列的和，只需要计算差值：
$$S([l,r])=S_r-S_{r-l}$$
就这样，通过 $O(n)$ 时间预处理，能够将单次查询区间和的复杂度降低到 $O(1)$．

参考实现：
```cpp
for(int i=1;i<=n;i++>)
{
    nums[i] += ps[i - 1];
}
```
