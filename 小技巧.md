## 一、优化

### 1.IO优化

#### 关闭同步流

```cpp
ios::sync_with_stdio(false);    // 解除C++流与C标准流的绑定
cin.tie(nullptr);               // 解除cin和cout的绑定
cout.tie(nullptr);
```
注意在使用了这串代码后，不能在程序中混用cin/cout和scanf/printf

#### 用`\n`替换`endl`

`std::endl` 不仅会换行，还会强制刷新输出缓冲区（flush）。在大量输出时，这会极其缓慢。所以最好使用`\n`替换`endl`

## 三、前缀和&差分

前缀和可以简单理解为「数列的前 $n$ 项的和」，是一种重要的预处理方式．对于长度为 $n$ 的序列 $\{ a_i\}$ ，如果要多次查询区间 $[l,r]$ 中序列数字的和，就可以考虑使用前缀和．

要想查询区间 $[l,r]$ 内的序列的和，只需要计算差值：
$$S([l,r])=S_r-S_{r-l}$$
就这样，通过 $O(n)$ 时间预处理，能够将单次查询区间和的复杂度降低到 $O(1)$．

参考实现：
```cpp
for(int i=1;i<=n;i++>)
{
    nums[i] += ps[i - 1];
}
```

### 二维前缀和

对于二维的前缀和，给定大小为 $m \times n$ 的二维数组 $A$ ，要求其前缀和 $S$ ，那么 $S$ 同样是大小为 $m \times n$ 的二维数组，且
$$S_{i,j} = \sum\limits_{i'\le i} \sum\limits_{j'\le j} A_{i',j'}$$

递推关系式：
$$ S_{i,j} = A_{i,j} + S_{i-1,j} + S_{i,j-1} - S_{i-1,j-1} $$

```cpp
for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
        {
            cin >> ps[i][j];
            ps[i][j] = ps[i][j] + ps[i-1][j] + ps[i][j-1] - ps[i-1][j-1];
        }
    }
```
查询左上角为 $(i_1,j_1)$ 、右下角为 $(i_2,j_2)$ 的矩阵的和：
$$ S_{i_2,j_2} - S_{i_1-1,j_2} - S_{i_2,j_1-1} + S_{i_1-1,j_1-1} $$

```cpp
ans = (ps[i2][j2] - ps[i1-1][j2] - ps[i2][j1-1] + ps[i1-1][j1-1]);
```
 
注意在具体实现时，矩阵的坐标和二维数组的索引位置不是对应的

## 四、最大/最小子数列和

kanade算法：核心思想是通过迭代数组的每个元素，维护两个变量来跟踪局部最优解和全局最优解。

具体实现(以最大子数列和为例)：
```cpp
int maxSubArray(int* nums, int numsSize) {
    int maxEndingHere  = nums[0], maxSoFar = nums[0];
    for (int i = 1; i < numsSize; i++) {
        maxEndingHere  = fmax(maxEndingHere  + nums[i], nums[i]);
        maxSoFar = fmax(maxSoFar, maxEndingHere );
    }
    return maxSoFar;
}
```

## 五、素数筛

埃氏筛法：该算法的（中文）名字“埃氏筛”是“埃拉托斯特尼筛法”的简称。

其基本思想是：
创建一个布尔（或整数）数组 `isPrime[0...n]`，初始时将所有从 2 到 n 的数都标记为“可能是素数”（例如 true 或 1）。
从第一个素数 p = 2 开始。
将 p 的所有倍数标记为“不是素数”（false 或 0）。
优化：我们从 p* p 开始标记，而不是 2* p。因为像 2* p, 3* p ...(p-1)*p 这样的数，它们已经被比 p 更小的素数（如 2, 3 等）给筛掉了。
找到下一个未被标记的数，将其设为新的 p（这个数必定是素数）。
重复步骤 3 和 4，直到 p*p 大于 n。
最后，数组中所有仍被标记为“可能是素数”的索引（下标）就是素数

```cpp
vector<int> primes;
vector<bool> numbers;

void isPrime(int m)
{
    numbers.assign(m+1,false);
    for(int i=2;i<=m;i++)
    {
        if(!numbers[i])
        {
            primes.push_back(i);
            for(int j=i*i;j<=m;j+=i)
            {
                numbers[j] = true;
            }
        }
    }
}
```
